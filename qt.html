Qt Designer Installation Guide

What is Qt Designer?

Qt Designer is a tool provided by the Qt framework that allows developers to design and build GUIs using a drag-and-drop interface. It generates .ui files that can be converted into Python code using tools like pyuic5 or pyuic6.

How to Install Qt Designer

  1. Go to the Qt Designer download page.
  2. Download the standalone installer for your operating system.
  3. Run the installer and follow the on-screen instructions.
  4. After installation, open Qt Designer and start designing your GUI!

Installing Qt Packages from Thonny

To use Qt Designer with Python, you need the PyQt5 or PyQt6 packages installed. Here's how you can do it using Thonny:

  1. Open Thonny IDE.
  2. Go to Tools > Manage Packages...

  3. In the search bar, type PyQt5 and Click Install.
  4. In the search bar, type PyQt5-tools and Click Install.

Optional: To convert .ui files to Python, install pyqt5-tools (for PyQt5 users) by typing this in the terminal: 

pip install pyqt5-tools

The converter tool will be located at:

.../Lib/site-packages/pyqt5_tools/Qt/bin/pyuic5.exe

Tips

  • Ensure Python and pip are installed before starting.
  • If you face errors during installation, try upgrading pip: pip install --upgrade pip
  • Keep Qt Designer and Python packages compatible (e.g., PyQt5 with Designer for Qt5).

Using Qt Designer Files in Python (PyQt5)

After designing your GUI in Qt Designer and saving it as a .ui file (e.g., power.ui), you can load and run it directly in Python using PyQt5 without needing to convert it manually.

Here's a basic example of how to load and run a UI file: 

from PyQt5.QtWidgets import* #import pyQt5 
from PyQt5.uic import *
app = QApplication([]) #create an application instance
fen = loadUi(\"name.ui\") #your interface name
fen.show() #to show the interface
fen.buttonName.clicked.connect(play)
app.exec_() #run the interface

This method is useful for quickly testing and iterating GUI designs without converting UI files manually.

Commonly Used Qt Functions in Python

Here are some essential functions you'll use often when working with PyQt5:

  • .text()
    Use this to get the text inside a QLineEdit. 
    user_input = lineEdit.text()
  • .setText("...")
    Use this to set text in a QLabel, QLineEdit, etc. 
    label.setText("Hello!")
  • .isChecked()
    Use this to check if a checkbox is checked (returns True or False). 
    if checkBox.isChecked():
      print("Checkbox is checked!")
  • .clicked.connect()
    This is how you link a button click to a Python function. 
    button.clicked.connect(my_function)

These functions help your GUI interact with the user and react to their input. You can access any widget using its objectName from Qt Designer, like self.lineEdit or just fen.lineEdit depending on your setup.
